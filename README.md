Used parts: 

- 5x Cherry MX Switches
- 1x SK6812 MINI Led
- 1x XIAO RP2040
- 5x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x EC11 Rotary encoder
- 1x 0.91 inch OLED display (SDD1306 OLED) 

Files: 

- KMK Firmware (main.py) 
- HACKPAD V3 (HACKPAD - BOTTOM - V3,HACKPAD - TOP - V3,HACKPAD - PCB - V3)
